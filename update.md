# Update Process

An update is:

1. Pull the latest code from the repository.
2. Rebuild the Android debug APK.
3. Overwrite the APK in Google Drive at:
   `/Users/radoslawchybicki/Library/CloudStorage/GoogleDrive-rchybicki@gmail.com/My Drive/phone share/CodexMonitor-debug.apk`

This keeps the shared install file current with the latest code.
